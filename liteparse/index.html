🔧 命令参考

支持的命令

命令说明
lit parse解析文档（PDF/DOCX/XLSX/PPTX/图片等）
lit screenshot生成文档页面截图
lit batch-parse批量解析多文档

常用用法

# 解析 PDF 为 Markdown
lit parse document.pdf > output.md

# 解析并指定格式为 JSON
lit parse document.pdf --format json

# 生成截图
lit screenshot document.pdf

# 批量解析
lit batch-parse ./input-dir/ --output ./output-dir/
