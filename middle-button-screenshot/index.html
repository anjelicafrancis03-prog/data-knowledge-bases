当前正式版本

已安装并由守护程序保活 仅限本机

正式版本使用鼠标右键：短按保持普通右键功能，长按约 0.5 秒后拖动框选区域，松开即保存 PNG 并复制到剪贴板。

启动与检查

正式版本由 Windows 守护程序和启动链自动运行。排障时先读取状态文件和应用日志，只按精确程序路径处理进程，不要批量结束 PowerShell。

powershell -ExecutionPolicy Bypass -File F:\codex\tools\right-button-screenshot-watchdog.ps1 -Once
命名说明

程序目录沿用早期名称 MiddleButtonScreenshot，旧 URL /middle-button-screenshot/ 为兼容已有链接而保留。产品名称和实际交互统一以“右键长按截图工具”为准。
