🎙️ Claude Code TTS 语音播报

Windows System.Speech 中文朗读 · 本地工具

📋 说明

利用 Claude Code Hooks + PowerShell + Windows System.Speech，在 Claude Code 回复时自动朗读中文。

所有处理在本地完成，不上传任何数据到云端。

💻 本机工具

控制面板（可视化开关）：

http://localhost:6120

或在终端执行：

%USERPROFILE%\.claude\hooks\voice-toggle.cmd
🔧 技术栈
  • PowerShell System.Speech API
  • Claude Code SessionStart/SessionEnd Hooks
  • 中文女声：Microsoft Huihui Desktop
  • 架构：tts-monitor.ps1 后台轮询 transcript → 提取 assistant 文本 → 朗读
📁 文件位置
%USERPROFILE%\.claude\hooks\
├── tts-start.cmd / tts-start.ps1    ← 启动钩子
├── tts-end.cmd / tts-end.ps1        ← 清理钩子
├── tts-monitor.ps1                   ← 轮询朗读守护
├── tts-functions.ps1                 ← 核心函数库
├── voice-toggle.cmd / .ps1           ← 一键开关
└── tts-state.json                    ← 开关状态
